Feature Engineering¶

Total Income Feature¶

In [53]:
df['Total_Income'] = (
    df['ApplicantIncome']
    + df['CoapplicantIncome']
)

Income Loan Ratio¶

In [54]:
df['Income_Loan_Ratio'] = (
    df['Total_Income']
    / df['LoanAmount']
)

EMI Feature¶

In [55]:
df['EMI'] = (
    df['LoanAmount']
    / df['Loan_Amount_Term']
)

Encode Categorical Variables¶

In [56]:
le = LabelEncoder()

categorical_cols = [

    'Self_Employed',
    'Property_Area',
    'Loan_Status',
